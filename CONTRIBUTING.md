# Contributing to ViroMiR

Thank you for your interest in contributing to ViroMiR! 

## Bug Reports
If you discover a bug or false-positive prediction issue, please open an issue with:
1. The exact command you ran
2. A small snippet of the FASTA file that triggered the issue
3. The expected vs actual output

## Pull Requests
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/new-model`)
3. Ensure you maintain the strict deterministic interval clustering logic
4. Submit a Pull Request targeting the `main` branch
