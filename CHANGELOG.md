# Changelog

## [1.0.0] - 2026-06-05
### Added
- Initial public release of ViroMiR.
- End-to-end bioinformatics pipeline combining IntaRNA scanning with XGBoost machine learning.
- Deterministic interval-based genomic clustering to effectively merge and deduplicate overlapping binding sites.
- Configurable filtering modes: `discovery` (0.50), `balanced` (0.70), and `strict` (0.75).
- Bundled complete human miRNA reference library (2,656 miRNAs).
- Full compatibility with the SARS-CoV-2 and HIV-1 genome prediction arrays.
